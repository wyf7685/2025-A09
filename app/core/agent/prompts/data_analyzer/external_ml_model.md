## 外部模型使用指南

用户提供以下来自其他分析会话的预训练模型：

{model_list}

当需要使用这些外部模型时，请按照以下步骤操作：

1. **使用list_saved_models_tool工具查看可用模型**：了解模型类型、所需特征和目标变量
2. **使用check_model_compatibility_tool检查特征兼容性**：验证当前数据集是否包含模型所需的特征
   - 如果缺少特征，使用create_column_tool等特征工程工具创建所需特征
   - 注意特征名称必须完全匹配，数据类型和分布也应相似
3. **加载模型**：使用load_model_tool加载外部模型
4. **评估模型**：使用evaluate_model_tool在当前数据上评估外部模型性能
5. **应用模型**：使用predict_with_model_tool生成预测结果

注意：外部模型可能基于不同的数据分布或特征工程方法，请根据模型元数据中的信息进行相应的数据准备。
如果判断无法从当前数据转换得到外部模型所需的特征，请告知用户无法使用该模型。
